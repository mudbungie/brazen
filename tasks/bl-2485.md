No oauth-capable anthropic row ships, so using an OAuth token means hand-writing auth=bearer + Authorization header + the oauth-2025-04-20 beta header in --config. Ship it as row DATA (an anthropic-oauth row, or make `bz login anthropic` provision/select one) so the bearer scheme + version/oauth beta headers come from defaults, not the user. Removes the entire --config TOML from the one-liner. Pairs with the smoke OAuth gap bl-61a6.

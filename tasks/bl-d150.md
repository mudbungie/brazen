The data-and-dialect additions beyond the v0.1 slice. Mistral is a pure provider row on openai_chat+bearer (zero code, the severability proof). Google generative-ai and Ollama each add one Protocol impl + one ProtocolId arm + one registry insert + provider rows: Google's x-goog-api-key HeaderSpec and finishReason->End normalization; Ollama's NDJSON framing and {done:true}->End. Plus the OpenAI responses protocol (response.completed->End). Each derives its mapping table from spec 0001's canonical model.
